# Asphalt8 Project
域名 www.asphalt8.net

###项目简介

Asphalt8是Gameloft的一款手机游戏，是iOS和android平台上最受欢迎的赛车游戏，简称A8。<br>
Asphalt8 Project是想构建一个专注于Asphalt8的小型社区，内容包括赛车统计，赛道地图，玩家社区和排名。
